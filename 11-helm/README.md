# Module 11: Helm - Kubernetes Package Management

> **Status**: Upcoming
> **Prerequisites**: [Module 10 - Kubernetes](../10-kubernetes/)

## Overview

Helm is the package manager for Kubernetes. This module covers charts, values files, templating, and release management - turning complex Kubernetes manifests into reusable, configurable packages.

## What Will Be Covered

- Helm architecture and concepts
- Charts, values, and templates
- Creating custom charts
- Chart repositories
- Release management and rollbacks

Content will be added as the course progresses.
