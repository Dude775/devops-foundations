# Module 12: Terraform - Infrastructure as Code

> **Status**: Upcoming
> **Prerequisites**: [Module 07 - AWS](../07-aws/)

## Overview

Terraform turns infrastructure into declarative code. This module covers providers, resources, state management, modules, and the IaC patterns that make infrastructure reproducible, versioned, and reviewable.

## What Will Be Covered

- Terraform fundamentals (init, plan, apply)
- Providers and resources
- State management and backends
- Variables and outputs
- Modules and reusability
- Best practices for production

Content will be added as the course progresses.
