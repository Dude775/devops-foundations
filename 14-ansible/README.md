# Module 14: Ansible - Configuration Management

> **Status**: Upcoming
> **Prerequisites**: [Module 02 - Linux](../02-linux/)

## Overview

Ansible automates configuration management across servers. This module covers inventory, playbooks, roles, and the concept of idempotency - ensuring that running the same automation twice produces the same result.

## What Will Be Covered

- Ansible architecture (agentless)
- Inventory and host management
- Playbooks and tasks
- Roles and reusability
- Variables and templates (Jinja2)
- Idempotency principles

Content will be added as the course progresses.
