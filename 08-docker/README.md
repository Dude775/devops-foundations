# Module 08: Docker - Containers and Images

> **Status**: Upcoming
> **Prerequisites**: [Module 06 - Python Advanced](../06-python-advanced/), [Module 07 - AWS](../07-aws/)

## Overview

Docker solves the "works on my machine" problem. This module covers containers, images, Dockerfiles, Docker Compose, and networking - the bridge between development and production that DevOps was built to manage.

## What Will Be Covered

- Container concepts vs virtual machines
- Dockerfiles and image building
- Docker Compose for multi-service applications
- Container networking and volumes
- Image registries
- Production Docker patterns

Content will be added as the course progresses.
