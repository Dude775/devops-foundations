# Module 00: Pre-Course - Web Fundamentals

> **Status**: Complete
> **Duration**: Self-paced
> **Prerequisites**: None

## Overview

Foundation in the three core web technologies - HTML, CSS, and JavaScript - completed through SoloLearn before the DevOps program began. These fundamentals are essential for working with web-based DevOps tools, understanding application structure for containerization, and debugging web applications in CI/CD pipelines.

## Learning Objectives

- [x] Understand HTML document structure and common elements
- [x] Apply CSS styling to HTML elements
- [x] Use JavaScript for basic interactivity and DOM manipulation

## Key Takeaways

- HTML defines structure, CSS defines appearance, JavaScript defines behavior
- These three technologies form the foundation of every web application a DevOps engineer will deploy, monitor, and troubleshoot
- Understanding the frontend stack helps debug issues that surface in CI/CD pipelines and production monitoring
