# Module 09: CI/CD - Pipelines and Automation

> **Status**: Upcoming
> **Prerequisites**: [Module 05 - Git & GitHub](../05-git-github/), [Module 08 - Docker](../08-docker/)

## Overview

CI/CD is the automated highway from code commit to production deployment. This module covers pipeline design, GitHub Actions, deployment strategies (blue-green, canary, rolling), and the automation principles that make Agile delivery possible.

## What Will Be Covered

- Continuous Integration concepts
- Continuous Delivery vs Continuous Deployment
- GitHub Actions pipelines
- Deployment strategies
- Artifact management
- Pipeline security

Content will be added as the course progresses.
