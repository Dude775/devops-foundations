# Module 13: ArgoCD - GitOps

> **Status**: Upcoming
> **Prerequisites**: [Module 09 - CI/CD](../09-ci-cd/), [Module 10 - Kubernetes](../10-kubernetes/)

## Overview

ArgoCD implements GitOps - using Git as the single source of truth for infrastructure and application state. This module covers GitOps principles, app-of-apps patterns, sync strategies, and continuous delivery to Kubernetes.

## What Will Be Covered

- GitOps principles and benefits
- ArgoCD architecture
- Application definitions
- Sync strategies and policies
- App-of-apps pattern
- Multi-environment management

Content will be added as the course progresses.
