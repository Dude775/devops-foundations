# Module 15: Jenkins - Build Automation

> **Status**: Upcoming
> **Prerequisites**: [Module 09 - CI/CD](../09-ci-cd/)

## Overview

Jenkins is the veteran CI/CD server still running in countless organizations. This module covers Jenkinsfiles, agents, shared libraries, and pipeline orchestration for complex build and deployment workflows.

## What Will Be Covered

- Jenkins architecture (controller, agents)
- Declarative vs scripted pipelines
- Jenkinsfile syntax
- Shared libraries
- Plugin ecosystem
- Integration with Docker and Kubernetes

Content will be added as the course progresses.
